feat: Implement AP Risk Rating System with vendor risk scoring

Adds comprehensive risk assessment for Accounts Payable vendors with
automated scoring, visual indicators, and filtering capabilities.

## Features

**Risk Scoring (0-100 points):**
- Missing ABN/Tax Number: +25 points
- Missing Tax Invoice: +20 points
- Unapproved Bill: +30 points
- Blocked Supplier: +60 points

**Risk Levels:**
- Low (0-19): Green badge
- Medium (20-44): Yellow badge
- High (45-69): Orange badge
- Critical (â‰¥70): Red badge

**Bank Change Detection:**
- Separate flag for bank account changes
- AlertTriangle icon with tooltip
- Helps identify potential fraud

## Database Changes

**xero_suppliers table:**
- Added tax_number (ABN/Tax ID)
- Added contact_status (ACTIVE/ARCHIVED/GDPRREQUEST)

- Added bank_account_name (null due to API limitation)
- Index on contact_status

**xero_bills table:**
- Added invoice_number (tax invoice)
- Added bill_bank_account_number (null due to API limitation)
- Added bill_bank_account_name (null due to API limitation)
- Index on invoice_number

## Known Limitations

**Xero API Constraints:**
- Bank account names NOT available (only numbers)
- Bill-specific bank accounts NOT available
- Tax number type not exposed (ABN vs EIN/SSN)

Documented in code comments and user documentation.
